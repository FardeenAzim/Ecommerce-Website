# Ecommerce Website

A full-stack e-commerce web application built using Django.

## Features

- User Authentication
- Product Catalog
- Shopping Cart
- Checkout
- Order Management
- Razorpay Payment Integration
- Admin Dashboard

## Technologies Used

- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite

## Installation

```bash
pip install -r requirements.txt
python manage.py runserver
